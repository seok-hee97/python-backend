# python-backend
### (Mini Twitter used by flask)



- Make python env used by conda

```
conda create --name api python=3.7
conda activate api
pip install -r requirements.txt
```



- Install httpie
```brew install httpie # MacOS```

```sudo apt install httpie # ubuntu```
