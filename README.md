# python-backend
### (Mini Twitter used by flask)



- Make python env used by conda

```
conda create --name api python=3.8
```

activate env
```
conda activate api
```

install python lib
```
pip install -r requirements.txt
```

deactivate env
```
conda deativate api
```


mysql 설정

```
mysql.server start
```
